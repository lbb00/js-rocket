# 定义变量

## var

声明一个变量

## let `ES6`

声明一个具有块级作用域的局部变量

### let 在 for 循环中的隐藏作用域

```javascript
// for(){ 循环体 } 在每次执行循环体之前，JS 引擎会把 i 在循环体的上下文中重新声明及初始化一次
for (let i = 0; i < 3; i++) {
  setTimeout(() => {
    console.log(i)
  }, 0)
} // -> 0、1、2

// 相当于
for (let i = 0; i < 3; i++) {
  let _i = i // 隐藏的_i
  setTimeout(() => {
    console.log(_i)
  }, 0)
}
```

## const `ES6`

声明一个具有块级作用域的常量

!> const 声明的常量并不是真正意义上的'常量'，

## var\let\const 对比

|                | var  | let  | const |
| -------------- | ---- | ---- | ----- |
| 不允许重复声明 | ×    | ✔️   | ✔️    |
| 块级作用域     | ×    | ✔️   | ✔️    |
| 暂时性锁区     | ×    | ✔️   | ✔️    |
| 初始化值       | 可选 | 可选 | 必选  |
